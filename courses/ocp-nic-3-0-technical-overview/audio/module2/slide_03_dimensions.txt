# pronunciation: SFF -> S F F; DSFF -> D S F F; 4C+ -> four C plus

The specification defines precise mechanical dimensions for each active form factor. S F F measures 76 millimeters wide by 115 millimeters deep, with an 11.50 millimeter height envelope and one four C plus connector. T S F F uses the same footprint and extends the height to 14.20 millimeters.

D S F F doubles the card width to 157.55 millimeters, keeps the 115 millimeter depth, and uses two connector positions for up to 32 P C I E lanes. T D S F F is the tall dual-width variant. For S F F and T S F F, an 89 millimeter pitch supports all three faceplate variants in the same platform. If thumbscrew support is not required, platform constraints can allow a smaller minimum pitch. D S F F pitch is 81.55 millimeters.
