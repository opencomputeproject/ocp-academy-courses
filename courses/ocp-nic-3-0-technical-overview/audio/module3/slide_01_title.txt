# pronunciation: PCIe -> P C I E; NIC -> Nick

Welcome to Module three: Electrical Interface. This module moves from mechanical fit into the connector signals, P C I E lane behavior, bifurcation, power states, and hot-swap considerations that let an OCP Nick three point zero card communicate with the host system.

The goal is to understand the interface as a set of coordinated signals, not just a long pin list. We'll pay special attention to the difference between mandatory normal power states and optional programming behavior.
