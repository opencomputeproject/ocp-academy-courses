# pronunciation: NIC -> Nick; FRU -> Fru; PCIe -> P C I E

OCP Nick three point zero is an open, standardized Nick specification purpose-built for hyperscale data centers. It defines mechanical, electrical, thermal, and management requirements in a single document. The result is a hot-swappable, field-replaceable unit, or Fru, that supports true multi-vendor interoperability.

The mental model is simple: a compliant card should work in a compliant baseboard slot. OCP Nick three point zero follows OCP Mezzanine two point zero, but it is not backward compatible because it uses a new connector and a new mechanical form factor. P C I E Gen six support is a capability of the current spec, not the reason older cards cannot plug in. The current release is version one point six point zero.
