# Module 2: The OCP Ecosystem & Governance

The second standalone Scrolling course in the **Intro to OCP** learning series. It introduces the people and organizations in the OCP community, the work they perform, the governance structure, and the contribution lifecycle.

## Lessons

1. Welcome, Introduction & Learning Objectives
2. Who is in the Community?
3. The Work of the OCP Community
4. How OCP is Organized
5. The Project Landscape
6. The Contribution Lifecycle
7. Knowledge Check
8. Summary

## Build

From the repository root:

```bash
./scripts/build-course.sh intro-to-ocp/module-2-the-ocp-ecosystem-governance
```

The source of truth is `course.json`. Course-owned learner media, captions, and the membership-benefits resource are retained under `resources/`. Generated SCORM files are not checked in.
