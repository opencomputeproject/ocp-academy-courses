# Module 1: The OCP Origins

The first standalone Scrolling course in the **Intro to OCP** learning series. It explains why the Open Compute Project began, how Open Rack evolved, and why open infrastructure matters to the market.

## Lessons

1. Welcome, Introduction & Learning Objectives
2. The OCP Origin Story
3. The Evolution of Open Rack
4. Why "Open" Matters (Market Impact)
5. Knowledge Check
6. Summary

## Build

From the repository root:

```bash
./scripts/build-course.sh intro-to-ocp/module-1-the-ocp-origins
```

The source of truth is `course.json`. Course-owned learner media and captions are retained under `resources/`. Generated SCORM files are not checked in.
