# Module 3: OCP Technologies (Open Rack & Cooling)

The third standalone Scrolling course in the **Intro to OCP** learning series. It examines Open Rack Version 3, the grid-to-chip architecture, the thermal challenge created by modern compute density, and advanced cooling solutions.

## Lessons

1. Welcome, Introduction & Learning Objectives
2. The ORv3 Deep Dive
3. Inside ORv3 Architecture
4. The Thermal Crisis
5. Cooling Solutions
6. Knowledge Check
7. Summary

## Build

From the repository root:

```bash
./scripts/build-course.sh intro-to-ocp/module-3-ocp-technologies-open-rack-cooling
```

The source of truth is `course.json`. Course-owned learner media and captions are retained under `resources/`. Generated SCORM files are not checked in.
