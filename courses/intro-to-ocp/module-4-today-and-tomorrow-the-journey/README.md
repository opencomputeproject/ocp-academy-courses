# Module 4: Today and Tomorrow (The Journey)

The fourth standalone Scrolling course in the **Intro to OCP** learning series. It helps learners turn the preceding concepts into action through the OCP hardware tiers, Marketplace, and adoption framework.

## Lessons

1. Introduction & Community Spotlight
2. Starting Your Journey
3. The OCP Hardware Tiers
4. Navigating the Marketplace
5. The Adoption Framework
6. Knowledge Check
7. Summary

## Build

From the repository root:

```bash
./scripts/build-course.sh intro-to-ocp/module-4-today-and-tomorrow-the-journey
```

The source of truth is `course.json`. Course-owned learner media and captions are retained under `resources/`. Generated SCORM files are not checked in.
