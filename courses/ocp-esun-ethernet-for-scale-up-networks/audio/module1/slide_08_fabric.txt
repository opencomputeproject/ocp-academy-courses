A lossless, differentiated, resilient fabric needs five things working together, and "ee-sun" names exactly which mechanism handles each. Transient congestion — short-lived microbursts caused by in-cast patterns — is handled at the link with priority flow control, PFC, the same mechanism standard data-center Ethernet uses. Ee-sun adds a second option, Credit-Based Flow Control, or CBFC, defined in the Ultra Ethernet Consortium specification. CBFC gives the receiver explicit credit-based admission, which lets switch designers use less buffer memory for the same lossless guarantee. Switches are required to support both PFC and CBFC. Persistent congestion — sustained back-pressure across the fabric, not just a single link — uses explicit congestion notification. Because there's no I P header to carry E C N, "ee-sun" puts it inside the "ee-sun" Header as EH-E C N. The two-bit encoding and the endpoint reaction match what operators already know from I P E C N, which keeps software stacks portable. Traffic differentiation — the way you give compute-critical traffic a separate class from background traffic — lives in EH-Co S: three bits, eight classes, no V Lan required. Resilience at the link level is handled by Link Level Retry, or LLR, again borrowed from the Ultra Ethernet specification. High-speed links have non-trivial bit error rates, and LLR lets the link recover from those errors locally without exposing every dropped frame to the end-to-end transport. Load balancing across a hierarchical topology uses the Flow Label in the EH as additional entropy for hashing — destination address, source address, and Flow Label, optionally more — keeping equal-cost paths balanced without random per-packet spraying that would break ordering. And finally, multi-hop loop avoidance falls to EH-T T L, the four-bit Time To Live in the "ee-sun" Header. Each switch decrements it; a value of zero causes the packet to be dropped — the same contract I P-T T L provides. Put it together and you have a fabric story that competes head-to-head with proprietary scale-up interconnects, on commodity Ethernet silicon.
