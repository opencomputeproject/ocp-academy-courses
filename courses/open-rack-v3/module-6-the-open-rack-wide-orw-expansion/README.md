# Module 6 - The Open Rack Wide (ORW) Expansion

The sixth standalone Scrolling course in the **Open Rack v3** learning series. It introduces the evolution from ORv3 to Open Rack Wide, including the ORW frame, cabling, transport, power shelves, and busbar.

## Lessons

1. Welcome, Intro & Learning Objectives
2. The Evolution of Open Rack: From ORv3 to ORW
3. Inside the ORW Frame
4. ORW Cabling and Transport
5. ORW Power Shelves & Busbar
6. Knowledge Check
7. Summary

## Review before packaging

The source of truth is `course.json`. The recovered learner media is under `resources/`, and `conversion.json` provides the conversion audit. A local, unzipped preview may be rendered for inspection; no SCORM zip should be created until the source is approved.

## Build after approval

From the repository root:

```bash
./scripts/build-course.sh open-rack-v3/module-6-the-open-rack-wide-orw-expansion
```

That command is intentionally deferred during source review because it creates the LMS-ready package and zip under `build/open-rack-v3/`.
