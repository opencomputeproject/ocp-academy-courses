# Module 7 - Advanced ORW Liquid Cooling & Practice

The seventh standalone Scrolling course in the **Open Rack v3** learning series. It covers ORW cooling and manifolds, liquid-cooling mechanics, implementation case studies, and the one-megawatt rack outlook.

## Lessons

1. Introduction & Learning Objectives
2. ORW Cooling & Manifolds
3. ORW Liquid Cooling Mechanics
4. ORW in Practice: Case Studies
5. Big Trends & The 1 Megawatt Future
6. Knowledge Check
7. Summary

## Review before packaging

The source of truth is `course.json`. The recovered learner media is under `resources/`, and `conversion.json` provides the conversion audit. A local, unzipped preview may be rendered for inspection; no SCORM zip should be created until the source is approved.

## Build after approval

From the repository root:

```bash
./scripts/build-course.sh open-rack-v3/module-7-advanced-orw-liquid-cooling-practice
```

That command is intentionally deferred during source review because it creates the LMS-ready package and zip under `build/open-rack-v3/`.
