# Module 4: ORv3 High Power Racks

The fourth standalone Scrolling course in the **Open Rack v3** learning series. It covers High Power Rack architecture, busbars, grounding, redundancy, cooling, safety, and support for high-density AI and HPC workloads.

## Lessons

1. Welcome, Intro & Learning Objectives
2. High Power Racks Overview
3. High Power Rack Technical Focus
4. Powering the HPR: Busbars, Grounding and Redundancy
5. HPR Cooling & Safety
6. Knowledge Check
7. Summary

## Review before packaging

The source of truth is `course.json`. The recovered learner media is under `resources/`, and `conversion.json` provides the conversion audit. A local, unzipped preview may be rendered for inspection; no SCORM zip should be created until the source is approved.

## Build after approval

From the repository root:

```bash
./scripts/build-course.sh open-rack-v3/module-4-orv3-high-power-racks
```

That command is intentionally deferred during source review because it creates the LMS-ready package and zip under `build/open-rack-v3/`.
