# Optical Connectivity & Reliability for AI Infrastructure

OCP Academy course sources for a learning-plan series on optical technologies, connectivity practices, and reliability workflows for AI infrastructure.

Each child folder is a standalone SCORM course source with its own `course.json`, narration scripts, figures, thumbnail, and course-owned teaching media. Course ordering is managed in the LMS learning plan, so folder names and learner-facing titles do not include fixed sequence numbers.

## Available Course Sources

| Folder | Course | Description |
|---|---|---|
| `introduction-to-emerging-optical-technologies` | Introduction to Emerging Optical Technologies | Emerging optical approaches for scale, efficiency, density, latency, and compute integration in AI infrastructure. |
| `enhanced-telemetry-optics` | Enhanced Telemetry Optics | Enhanced telemetry for finding what failed, where, and why in AI optical fabrics. |
