# pronunciation: CMIS -> C M I S; CDB -> C D B; I2C -> I squared C; SONiC -> sonic; PMD -> P M D; VDM -> V D M; DDMI -> D D M I; SNR -> S N R; PAM4 -> PAM four; MPI -> M P I; FEC -> feck; BER -> B E R; LOS -> L O S; LOL -> L O L; SDK -> S D K; API -> A P I
모듈 4는 부품들을 배치에 연결한다. 텔레메트리는 반복 가능한 운영자 워크플로우가 될 때 가장 가치가 있습니다: 적절한 엔드포인트를 조회하고, 현재 상태를 조회하며, 지속 가능한 이벤트 이력을 검사하고, 펌웨어를 안전하게 업데이트하며, 유용한 데이터를 네트워크 전체 상태 뷰로 전송합니다.

우리는 역량과 워크플로우에 집중할 것입니다. 상세한 명령 테이블은 사양 내에 남아 있어야 할 자리에 남아 있습니다.
이 마지막 모듈에서 신뢰성이 생명주기가 됩니다. 장애 진단을 돕는 동일한 인터페이스가 유지보수, 버전 관리, 실제 사고 시 사용하는 호스트 도구도 지원해야 합니다.
