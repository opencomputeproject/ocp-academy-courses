# pronunciation: CMIS -> C M I S; CDB -> C D B; I2C -> I squared C; SONiC -> sonic; PMD -> P M D; VDM -> V D M; DDMI -> D D M I; SNR -> S N R; PAM4 -> PAM four; MPI -> M P I; FEC -> feck; BER -> B E R; LOS -> L O S; LOL -> L O L; SDK -> S D K; API -> A P I
모듈 3은 측정된 것에서 호스트가 그것을 어떻게 얻는지로 넘어갑니다. 이 명세는 호스트와 모듈 간의 명령-응답 교환을 위해 C D B, 즉 명령 데이터 블록을 사용합니다.

우리는 레벨을 개념적인 수준으로 유지할 것입니다. 준비, 트리거, 폴링, 수집 워크플로우를 이해해야 합니다. 또한 상태 확인, 이벤트 로그, 원격 C M I S 접근, 페이로드 처리가 신뢰할 수 있는 운영에 왜 중요한지 이해해야 합니다.
모듈 2가 어휘라면, 이 모듈은 문법입니다. C D B는 호스트 소프트웨어가 데이터를 반복적으로 요청하고 결과를 자신 있게 해석할 수 있도록 요청과 응답이 어떻게 구조화되는지 정의합니다.
