# pronunciation: CMIS -> C M I S; CDB -> C D B; I2C -> I squared C; SONiC -> sonic; PMD -> P M D; VDM -> V D M; DDMI -> D D M I; SNR -> S N R; PAM4 -> PAM four; MPI -> M P I; FEC -> feck; BER -> B E R; LOS -> L O S; LOL -> L O L; SDK -> S D K; API -> A P I
API host SDK và dòng lệnh là nơi thông số kỹ thuật trở nên hữu ích cho người vận hành. Thông số kỹ thuật liệt kê các danh mục ánh xạ trực tiếp tới quy trình tại hiện trường.

Truy cập C M I S tiêu chuẩn cho phép phần mềm đọc trạng thái mô-đun cục bộ. Truy cập C M I S từ xa cho phép phần mềm trích xuất dữ liệu tiêu chuẩn từ điểm cuối ở xa. Tải xuống từ xa hiển thị các tham số mô-đun cục bộ hoặc từ xa. Các lệnh nhật ký sự kiện đọc các sự kiện gần đây, xóa nhật ký và điều chỉnh độ trễ. Các lệnh firmware quản lý quy trình cập nhật cục bộ hoặc từ xa.

Thông số kỹ thuật cũng bao gồm khả năng quan sát điểm tình trạng liên kết mô-đun và thời gian kỷ nguyên. Những danh mục đó quan trọng vì chúng kết nối chuỗi bằng chứng: giá trị hiện tại, sự kiện lịch sử, tương quan thời gian và điểm tình trạng ngắn gọn. Một API tốt giúp chuỗi bằng chứng đó trở nên hữu dụng trong quá trình đánh giá chất lượng và trong các sự cố.
Các danh mục API cũng phải liên kết tới tài liệu và đào tạo. Người vận hành nên biết lệnh nào truy xuất dữ liệu đo từ xa hiện tại, lệnh nào truy xuất các sự kiện gần đây, lệnh nào xóa lịch sử và lệnh nào thay đổi hành vi.
