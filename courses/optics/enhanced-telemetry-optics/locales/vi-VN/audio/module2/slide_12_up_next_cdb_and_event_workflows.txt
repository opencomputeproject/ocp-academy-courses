# pronunciation: CMIS -> C M I S; CDB -> C D B; I2C -> I squared C; SONiC -> sonic; PMD -> P M D; VDM -> V D M; DDMI -> D D M I; SNR -> S N R; PAM4 -> PAM four; MPI -> M P I; FEC -> feck; BER -> B E R; LOS -> L O S; LOL -> L O L; SDK -> S D K; API -> A P I
Làm tốt lắm. Bạn đã hoàn thành lớp số liệu. Giờ đây, bạn có thể nhóm dữ liệu đo từ xa thành chất lượng tín hiệu, lỗi, quá trình khôi phục, môi trường và danh tính, đồng thời bạn biết lý do tại sao điểm tình trạng nên duy trì ở danh mục yếu nhất.

Tiếp theo: Quy trình làm việc C D B, nhật ký sự kiện và truy cập từ xa. Chúng ta sẽ xem xét cách các lệnh di chuyển qua giao diện mô-đun, cách lưu giữ lịch sử sự kiện và cách phần mềm host tiếp cận dữ liệu C M I S ở xa thông qua mô-đun cục bộ.
