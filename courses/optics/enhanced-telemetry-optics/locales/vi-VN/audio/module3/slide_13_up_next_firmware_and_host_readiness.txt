# pronunciation: CMIS -> C M I S; CDB -> C D B; I2C -> I squared C; SONiC -> sonic; PMD -> P M D; VDM -> V D M; DDMI -> D D M I; SNR -> S N R; PAM4 -> PAM four; MPI -> M P I; FEC -> feck; BER -> B E R; LOS -> L O S; LOL -> L O L; SDK -> S D K; API -> A P I
Làm tốt lắm. Bạn đã hoàn thành mô-đun quy trình lệnh-phản hồi và sự kiện. Bây giờ bạn có thể theo dõi trình tự C D B, giải thích lý do tại sao việc kiểm tra trạng thái lại quan trọng và mô tả cách nhật ký sự kiện và quyền truy cập C M I S từ xa lưu giữ bằng chứng.

Tiếp theo: firmware, phần mềm host và mức độ sẵn sàng triển khai. Chúng ta sẽ kết nối giao diện đo từ xa với các hoạt động trong vòng đời, API host và các hoạt động kiểm tra giúp việc triển khai trở nên đáng tin cậy trên quy mô lớn.
